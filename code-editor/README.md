# COZIP - Online Code Editor

COZIP is a lightweight online HTML, CSS, and JavaScript code editor designed for beginners and quick practice sessions.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- **Integrated Code Editor:** A ready-to-use environment for writing and testing code.
- **Interactive Theme:** Designed to provide an efficient and engaging coding environment for beginners.
- **Optimization Settings:** Customize your workspace with adjustable font sizes and font families.
- **Live Preview:** View your code's output instantly without refreshing the page.
- **Run Button:** Click the run button to view the preview without manual browser refreshes.
- **localStorage:** COZIP uses localStorage to store user preferences and code editor state.

## Limitations

- **Project Scale:** Not intended for building large-scale projects.
- **JavaScript Support:** Limited support for complex JavaScript; heavy scripts may affect editor performance.
- **File Management:** Uses fixed built-in files for `.html`, `.css`, and `.js`. Creating new files or folders is not supported.

## Best Use Cases

COZIP is preferred for:

- Small practice tasks and exercises.
- Experimenting with HTML, CSS, and basic JavaScript.

## Getting Started

To start coding, open `code.html` in your browser or navigate through the `index.html` landing page.
