# Checkers Game

## Description

Offline 2 player checkers game built using HTML, CSS and JavaScript.

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Play

1. Open `index.html` in any modern web browser.
2. Players take turns moving their pieces diagonally to an adjacent unoccupied dark square.
3. To capture an opponent's piece, jump over it into an empty square directly behind it.
4. If a piece reaches the opposite end of the board, it is promoted to a **King**, allowing it to move and capture both forward and backward.
5. The game ends when a player captures all of their opponent's pieces or blocks them from making any further moves.

## Features

- **Local Multiplayer**: Play with a friend on the same machine.
- **King Promotion**: Automatic logic for crowning pieces.
- **Move Validation**: Visual feedback for legal moves and selection.
- **Responsive UI**: Clean interface built with CSS Flexbox/Grid.
