# 2D Bike Game

## Description

A 2D bike game where the player controls a bike and avoids obstacles.

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Play

- Use Space to jump
- Avoid the obstacles to score points.
- The game ends when the bike collides with an obstacle.
