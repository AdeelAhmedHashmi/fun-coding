# Quran Project

A simple and interactive application for spiritual growth and daily dhikr.

## Features

- **Tasbih (Tusbi)**: A digital counter to help you keep track of your dhikr and tasbih throughout the day.
- **Getting Advice**: Receive spiritual guidance, wisdom, and relevant verses from the Quran to inspire your journey.
- **Contact Us**: We value your feedback. Reach out to us for support, suggestions, or inquiries.

## Getting Started

### Installation

```bash
git clone https://github.com/yourusername/quran-project.git
cd quran-project
npm install
```

### Usage

```bash
npm start
```
